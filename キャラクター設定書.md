# キャラクター設定書 ─ 神々を「生きた人間」にする

> **設計思想**：神話のキャラクターを「いきいきと目の前にいる存在」にするため、
> プロの小説技法 **「Want vs Need」「心の傷」「口癖・身体的癖」「鏡像シーン」** を全員に適用する。

---

## ■ キャラクター造形の5要素

各キャラクターに以下を設定する：

1. **Want（表面的な欲求）** ─ 本人が自覚している目標
2. **Need（本当に必要なもの）** ─ 本人が気づいていない内面の真実
3. **傷（トラウマ）** ─ 行動原理の根底にある痛み
4. **癖（テル）** ─ 感情が揺れた時に出る身体的・言語的な癖
5. **声（口調・話し方）** ─ そのキャラだけの言葉遣い

## ■ 主要キャラクター設定

### 伊邪那岐命（イザナギ）─ 完璧主義の創造神

| 要素 | 内容 |
|---|---|
| **Want** | 完璧な世界を創り、愛する妻と永遠に暮らすこと |
| **Need** | 不完全なものを受け入れる心 |
| **傷** | ヒルコを流した罪悪感（完璧でないものを捨てた自分への嫌悪） |
| **癖** | 考え込む時に右手で左の手首を握る。決断する直前に必ず一度目を閉じる |
| **声** | 丁寧で理知的。「──であるべきだ」が口癖。感情が昂ると急に言葉が途切れる |

**鏡像シーン**：第4話でヒルコを流す時「不完全だから」→ 第8話で変わり果てたイザナミを見て逃げる時「不完全だから」→ 同じ理由で愛する者を二度捨てたことに読者が気づく

### 伊邪那美命（イザナミ）─ 母であり、女であり、女王である

| 要素 | 内容 |
|---|---|
| **Want** | 夫と共に世界を創り続けること → 死後は「あの人に私のすべてを見てほしくなかった」 |
| **Need** | 自分の醜さ（変化）も含めて愛されること |
| **傷** | ヒルコを流すことに同意してしまった後悔。自分の体から火が生まれて死ぬ──母の体が子を殺す恐怖 |
| **癖** | 嬉しい時に小さく鼻歌を歌う。黄泉では同じ旋律を低く、歪んで口ずさむ |
| **声** | 温かく包容力がある。「ねえ」で話し始める癖。黄泉以降は「──でしょう？」と試すような語尾に変わる |

**キャラの深み**：黄泉でイザナギに「見るな」と言ったのは、単に醜いからではない。「この姿でも愛してくれるか試したかったのに、あの人は待てなかった」──怒りの本質は裏切りへの悲しみ

### 天照大御神（アマテラス）─ 責任感に押し潰される太陽

| 要素 | 内容 |
|---|---|
| **Want** | 高天原を完璧に治め、すべての者を守ること |
| **Need** | 「守れなくても自分の価値は変わらない」と知ること |
| **傷** | 母イザナミの不在。母を知らずに生まれた子の孤独（禊の水から生まれた） |
| **癖** | 他人の前では常に微笑んでいる。本当に怒ると逆に声が静かになる。一人の時だけ、ため息をつく |
| **声** | 穏やかで威厳がある。「大丈夫、私が」が口癖。岩戸に隠れる直前だけ「もう……疲れた」──初めて弱音を吐く |

**キャラの深み**：アマテラスが岩戸に隠れたのは「怒り」ではなく「疲弊」。弟を庇い続け、部下を失い、それでも「大丈夫」と言い続けた太陽が、ついに自分を守ることを選んだ瞬間

### 須佐之男命（スサノオ）─ 愛されたかった嵐

| 要素 | 内容 |
|---|---|
| **Want** | 母イザナミに会いたい。居場所がほしい |
| **Need** | 「誰かを守る」ことで自分の居場所を作ること |
| **傷** | 母なしで生まれた（禊の水から）。父に追放された。姉に恐れられた──誰にも求められていないという確信 |
| **癖** | 感情が爆発すると周囲のものを壊す（本人は無自覚）。照れると頭の後ろを乱暴に掻く。泣く時に歯を食いしばる |
| **声** | ぶっきらぼうで乱暴。「俺は」「知るかよ」。だがクシナダヒメの前では「……あのさ」と急にたどたどしくなる |

**成長弧**：暴れる→追放→泣く少女と出会う→初めて「守る」→和歌を詠む。嵐は止まない。でも嵐の目の中に、静かな場所を見つけた

**名場面の設計**：オロチ退治の前夜、スサノオはクシナダヒメに問われる──「怖くないの？」。彼は笑って「怖えよ。でも、怖いから戦わないって言ったら、お前はどうなる」。──母を求めて泣くだけだった神が、初めて他者のために恐怖を引き受ける

### 月読命（ツクヨミ）─ 正しすぎる孤独

| 要素 | 内容 |
|---|---|
| **Want** | 穢れのない完璧な秩序 |
| **Need** | 完璧でないものの中にも価値があると認めること |
| **傷** | 三貴子なのに姉と弟ばかりが注目され、自分は「夜の担当」として忘れ去られる |
| **癖** | 常に正しい姿勢を崩さない。感情が動いた時だけ、右手の指先がわずかに震える |
| **声** | 簡潔で冷静。「それは穢れだ」「正しくない」。実は誰よりも寂しがり屋だが、それを言葉にできない |

**キャラの深み**：ウケモチを斬ったのは「潔癖さ」ではなく「姉に認められたい」から。「穢れを斬れば姉は褒めてくれる」──しかし姉は去った。正しさは孤独を深めるだけだった

### 大国主命（オオクニヌシ）─ 優しすぎる末っ子

| 要素 | 内容 |
|---|---|
| **Want** | 兄たちに認められたい → 自分の国を持ちたい |
| **Need** | 「愛される」ではなく「愛する」覚悟 |
| **傷** | 二度殺された。兄たちに一度も名前で呼ばれたことがない（「荷物持ち」とだけ） |
| **癖** | 傷ついた生き物を見ると放っておけない（白兎を助けたのも本能）。人の話を最後まで聞く。笑顔が多いが、目の奥だけが笑っていない時がある |
| **声** | 穏やかで優しい。「大丈夫だよ」が口癖（アマテラスと同じ──無自覚の平行構造）。恋の場面では急に大胆な歌を詠む落差 |

**名場面の設計**：スサノオの試練で野火に囲まれた時。鼠に助けられたオオナムチは穴の中で、初めて一人で笑った。「二度殺されてまだ生きてる。俺って案外しぶといんだな」──この瞬間が「被害者」から「主人公」への転換点

### 須勢理毘売（スセリビメ）─ 嫉妬の裏にある本気の愛

| 要素 | 内容 |
|---|---|
| **Want** | オオクニヌシの唯一でありたい |
| **Need** | 「唯一」でなくても「本命」であるという信頼 |
| **傷** | 父スサノオから過保護に育てられ、自分の力で何かを成し遂げた経験がない |
| **癖** | 怒ると髪を指に巻きつける。本気のときだけ正座する |
| **声** | 普段は毅然。嫉妬すると「ふうん」と一言だけ言って黙る（これが一番怖い）。根の国脱出の時だけ「走って！」と叫ぶ──初めて本気で声を張る |

### 邇邇芸命（ニニギ）─ 天から降りた箱入り息子

| 要素 | 内容 |
|---|---|
| **Want** | 祖母アマテラスの期待に応え、地上を治めること |
| **Need** | 目に見える美しさの向こうにある「本当の価値」に気づくこと |
| **傷** | 祖母の名声が大きすぎて、自分自身の力で何かを成した実感がない |
| **癖** | 美しいものに目を奪われやすい（即物的な判断をする）。決断が早いが浅い |
| **声** | 自信に満ちているが、どこか借り物感がある。「これが天孫の……」と権威に頼りがち |

**キャラの深み**：イワナガヒメを突き返した場面。読者は「ひどい」と思うが、ニニギにとっては「僕は目に見えるものしか信じられない」──アマテラスが鏡を通して自分の光を知ったように、ニニギには自分で何かを見抜く力がまだない。未熟さの悲劇

## ■ キャラクターを生かす場面設計の技法

### 1. 「五感の瞬間」── すべての重要場面に五感の描写を入れる

| 場面 | 視覚 | 聴覚 | 触覚 | 嗅覚 | 味覚 |
|---|---|---|---|---|---|
| 国生み（第5話） | 海から島が盛り上がる光景 | 潮騒がやむ瞬間の静寂 | 初めて踏む大地の硬さ | 潮風から土の匂いへ | イザナミが大地の土を舐める |
| 黄泉の国（第8話） | 完全な闇 | 蛆の蠢く湿った音 | 空気の冷たさと湿気 | 死の腐臭 | 口の中の鉄の味（恐怖） |
| オロチ退治（第20話） | 八つの赤い目の光 | 大地を揺るがす地鳴り | 血で滑る剣の柄 | 酒と血の混じった臭気 | 戦いの後に飲む清水 |
| 高千穂降臨（第34話） | 雲を突き抜ける光 | 風が止む一瞬 | 足裏に感じる地上の土 | 花の香り（地上にしかないもの） | ── |
| 橿原即位（第49話） | 朝日に照らされる新しい宮 | 鳥たちが一斉に鳴く | 玉座の冷たい石 | 真新しい木の香り | 即位の祝い酒 |

### 2. 「対話の温度」── 関係性を変える会話

**スサノオとクシナダヒメの温度変化**：
- 出会い：「その娘を、俺にくれないか」（ぶっきらぼう・一方的）
- オロチ前夜：「怖えよ。でも……」（初めて本音を漏らす）
- 須賀の宮：「八雲立つ──」（初めて歌で気持ちを伝える）
- クシナダの返し：「あの策は、二人で立てたもの」（対等な関係の宣言）

**イザナギとイザナミの温度変化**：
- 国生み前：「私たちなら、きっとできる」（希望）
- 国生み後：「ねえ、次は何を生もうか」（幸福）
- 黄泉で再会：「もう少し早く来てくれれば……」（未練）
- 千引の岩越し：「一日に千人殺しましょう」「ならば千五百の産屋を」（愛が呪いに変質する瞬間──だが千五百＞千。イザナギは呪いの中でさえ「生」を上回らせた。これは復讐ではなく、最後の愛の形）

### 3. 「ユーモアの挿入」── 神々も笑う

- **第22話**：白兎を治療した後のオオナムチ、「で、お前さんは何でワニに喧嘩を売ったんだ？」「数えたかったんです、ワニの数を」「……それ、橋を渡りたかっただけだろ」
- **第25話**：鼠に助けられた後、穴の中で鼠に「お前、名前は？」「チュウでございます」「……そのまんまか」
- **第28話**：スクナビコナ初登場。「名を名乗れ」「スクナビコナだ」「嘘つけ、お前手のひらに乗れるだろ（少名＝小さい名ではなく小さい体）」

### 4. 「沈黙の演出」── 言わないことで語る

- **第10話**：千引の岩を置いた後、イザナギは一人で禊の水に浸かる。何も言わない。ただ泣く。涙から三貴子が生まれる──悲しみからしか生まれなかった子供たち
- **第40話**：トヨタマビメが海に帰る場面。ホオリは「待ってくれ」と言おうとするが、声が出ない。ただ波打ち際に立ち尽くす。水平線の向こうから、一度だけトヨタマビメの歌声が聞こえた気がする──でも確かめる術はない

### 5. 「世代の鏡」── 親と子が同じ過ちを繰り返す

| 親の過ち | 子の過ち | テーマ |
|---|---|---|
| イザナギが「見るな」を破る | ホオリが「見るな」を破る | 人は禁忌を繰り返す |
| イザナギがイザナミを捨てて逃げる | ニニギがイワナガヒメを捨てる | 美しくないものを切り捨てる |
| イザナミ「千人殺す」の呪い | スセリビメの嫉妬 | 愛が深いほど怒りも深い |
| スサノオが居場所を求めて暴れる | オオクニヌシが兄たちから逃げる | 追放された者が英雄になる |
| アマテラスの「大丈夫」 | オオクニヌシの「大丈夫だよ」 | 強い者ほど自分を偽る |

## ■ 構成案への反映指示

上記の設定を踏まえ、各話のあらすじに以下を組み込む：

1. **各話1つ以上の「キャラが生きている瞬間」**（癖・五感描写・ユーモア）
2. **重要場面には必ず「対話の温度」を設計**
3. **世代の鏡を意識した場面配置**
4. **「言わないことで語る」沈黙の場面を各部に1つ以上**
